# Recall
a app that convert source material into flashcards, memory table and practice quizzes

This app is for people who dont like boring notebooks and textbooks but find fancy graphics and animations are much better way to learn,
the core func is you give your material whether than pdf, videos, audio etc and it converts into flashcard and other form of media that can help you study better without the fear of trying to finshi a 2 page long text
